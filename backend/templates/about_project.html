AI-Powered Data Analytics Platform with NLP Chatbot

National Institute of Electronics & Information Technology (NIELIT), Srinagar

Prepared by: Waqar Maqbool Wani (23045136010) & Inshpreet Kour Mehta (23045136011)

Introduction

The proposed project provides users with a seamless and intuitive way to analyze datasets and extract meaningful insights through natural language interaction. Users can upload datasets in various formats (e.g., CSV, Excel, JSON), and the system automatically generates descriptive statistics, visualizations, and actionable insights. It integrates an NLP chatbot powered by APIs such as ChatGPT, DeepSeek, or similar, allowing users to interact with their data through plain English queries, with the chatbot responding with relevant summaries, charts, or dashboards.

Key Features

  • Data Upload & Processing (CSV, Excel, JSON formats).
  • Automated Analysis: Summary statistics, visualizations, pattern and anomaly detection.
  • NLP Chatbot: Natural language queries via AI APIs like ChatGPT, DeepSeek.
  • Interactive Dashboard: Real-time filtering, sorting, export options using Plotly Dash or Streamlit.

Use Cases

  • Healthcare: Analyze patient data to detect disease trends and predict risks.
  • Education: Monitor student performance and identify at-risk learners.
  • Business: Visualize sales data, customer demographics, and manage inventory efficiently.

Problem Formulation

Organizations generate vast amounts of data, but non-technical users find it difficult to derive insights due to the technical barrier. Traditional tools like Tableau and Power BI lack natural language interaction. General-purpose AI models are not secure for sensitive data processing.

Proposed Solution

  • Automated Data Analysis
  • NLP Chatbot Integration
  • Secure Local Deployment for Privacy

Objectives

  • Enable Seamless Data Upload and Processing
  • Provide Automated Insights and Visualizations
  • Integrate NLP-Based Querying Capabilities
  • Ensure Data Privacy and Security
  • Support Domain-Specific Use Cases
  • Promote Scalability and Flexibility

Methodology/Planning of Work

  • Frontend: React.js or Streamlit - File upload, dashboards, chatbot UI
  • Backend: Flask or Django - API handling, authentication
  • Data Processing: Pandas, PySpark - Cleaning, preprocessing
  • Analytics Engine: Plotly, Seaborn - Visualizations, anomaly detection
  • NLP Chatbot: ChatGPT, DeepSeek - Query interpretation
  • Database: SQLite, PostgreSQL, MongoDB - Storage and backup
